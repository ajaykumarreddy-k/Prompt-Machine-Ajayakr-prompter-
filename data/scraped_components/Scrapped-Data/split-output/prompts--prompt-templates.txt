# Prompt Templates

## What Makes a Strong UI Prompt
1. **Explicit Identification**: Strict taxonomy of the exact target demographic (B2B, consumer) and product shell parameters up-front.
2. **Structural Mapping**: Calling out unequivocally what distinct layout components appear and in what specific cascading vertical rhythm.
3. **Palette Determination**: Clearly declaring the isolated Primary, Secondary, and Accent vectors utilizing strict Tailwind identifier terminology.
4. **Keyword Binding**: Deploying heavy categorical adjectives ("Corporate", "Brutalist", "Glassmorphic") commanding total system aesthetics.
5. **Data Expectations**: Declaring strictly what dummy matrix rows, avatar counts, or table routing variables must manifest.
6. **Responsiveness Flags**: Routinely demanding absolute mobile grid collapsing protocols avoiding overflow bleeds.
7. **Animation Protocols**: Calling out exact singular visual interactive keyframe hits rather than submitting vague "make it move" triggers.
8. **Component Referencing**: Calling out explicit files (`[cards]`, `[navbar]`) from the knowledge base directory schemas to enforce strict compliance mapping.
9. **Spacing Authority**: Specifically enforcing vertical `py-32` blocks to avoid claustrophobic code generation mappings.
10. **Stack Lockdown**: Reinforcing React hooks and Tailwind CSS utility restraints universally.
