# Prompt Templates

## What Makes a Strong UI Prompt
1. **Explicit Identification**: Strict naming of exact target demographic and product type up-front.
2. **Structural Mapping**: Calling out exactly what components appear and in what vertical rhythm.
3. **Palette Determination**: Clearly declaring Primary, Secondary, and Accent vectors using precise terminology.
4. **Keyword Binding**: Deploying heavy descriptive nouns (e.g., "SaaS", "Glassmorphism") determining total visual attitude.
5. **Data Expectations**: Declaring strictly what dummy data rows or table attributes must manifest.
6. **Responsiveness Flags**: Consistently demanding mobile grid collapse behavior explicitly.
7. **Animation Protocols**: Calling out exact singular visual interaction moments instead of generalized "make it move" requests.
8. **Stack Lockdown**: Reinforcing React and Tailwind generation restraints strictly to prevent styling drift.
