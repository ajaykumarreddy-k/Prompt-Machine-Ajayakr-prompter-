# UI Prompt — Code Assistant Evaluation Framework

Code Assistant Evaluation Framework — landing page
Target audience: Developers and Researchers

Key features:
  - Evaluates language model's ability to understand codebase
  - Tests code location, explanation, system understanding, and edge case reasoning
  - Detects hallucinations and provides accuracy scores

Layout:
  - Navbar
  - Hero
  - Cards
  - Footer

Design:
- Color palette: Primary blue-600 Secondary slate-800 Accent indigo-500
- Background: slate-50  Text: slate-900
- Style: code, evaluation, model, accuracy
- Tone: technical
- Responsive: yes (mobile-first, grid collapses)

Stack: React functional components, Tailwind CSS utility classes only
