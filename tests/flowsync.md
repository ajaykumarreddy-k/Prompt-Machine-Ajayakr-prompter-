# FlowSync PRD

## Product Overview
FlowSync is a modern SaaS platform for task management, team collaboration, and workflow automation. It targets high-performance teams.

## Key Features
- Dynamic task boards with real-time syncing.
- Team collaboration tools (chat, comments, shared docs).
- Workflow automation engine with visual builder.
- Detailed analytics and reporting.

## Style & Tone
Professional, clean, and high-performance. Use a zinc-based minimal editorial style with red accents.

## Technical Requirements
- React with Tailwind CSS.
- React Router for multi-page navigation.
- Strict design token enforcement.
- Mobile-first responsive design.
